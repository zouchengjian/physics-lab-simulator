# Physics Lab Simulator

## Project Documentation

### Overview
The Physics Lab Simulator is an interactive application that allows users to conduct virtual physics experiments. It covers various topics in physics, including mechanics, thermodynamics, and electromagnetism.

### Features
- User-friendly interface for conducting experiments.
- Comprehensive simulation of physical phenomena.
- Detailed instructions and explanations for each experiment.

### Installation
To install the Physics Lab Simulator, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/zouchengjian/physics-lab-simulator.git
   ```
2. Navigate to the project directory:
   ```bash
   cd physics-lab-simulator
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```

### Usage
To run the simulator, use the following command:
```bash
npm start
```
